DataFusion-RT нужно запускать только из распакованной папки целиком.

Нельзя:

- копировать один `DataFusion-RT.exe` отдельно
- запускать `exe` прямо из zip-архива
- удалять соседние `.dll`, `base_library.zip`, папку `config`

Правильно:

1. распаковать весь архив
2. открыть папку `DataFusion-RT`
3. запускать `DataFusion-RT.exe` только там
4. папку `build-win7-x86` не использовать вообще

Если видите ошибку загрузки `python38.dll` или другой DLL:

- проверьте, что рядом с EXE лежит весь комплект файлов
- проверьте, что рядом лежат `ucrtbase.dll` и `api-ms-win-crt-*.dll`
- не запускайте из окна zip
- не переносите один exe без папки
